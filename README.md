## Tech Stack

- **Styling**: Tailwind CSS v4
- **Deployment**: GitHub Pages via GitHub Actions
- **Features**: Dark mode, responsive design, static export

## Sections

- Featured Projects
- Research & Publications
- Experience
- Achievements
- Technical Arsenal (Skills)
- Journey Timeline

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Deployment

Automatically deploys to GitHub Pages on push to `main` branch via GitHub Actions workflow.
